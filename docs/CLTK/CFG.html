    class CLTK::CFG

    • CLTK::CFG
    • Reference
    • Object

    Overview

    The CFG class is used to represent context-free grammars. It is used by the CLTK::Parser class to represent the parser's grammar, but can also be used to manipulate arbitrary CFGs.

    Defined in:

    cltk/cfg/production.cr
    cltk/cfg/item.cr
